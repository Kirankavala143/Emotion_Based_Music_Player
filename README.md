# 🎧 Emotion Based Music Player

An AI-powered web application that detects user emotions in real-time and plays music accordingly.

---

## 🚀 Features

* 🎥 Real-time face detection using webcam
* 😊 Emotion recognition (Happy, Sad, Angry, Neutral, Surprise)
* 🎵 Automatic music recommendation based on emotion
* ⚡ Fast and responsive UI
* 🌐 Web-based interface using Flask

---

## 🧠 Tech Stack

* Python
* Flask
* OpenCV
* DeepFace (CNN-based emotion detection)
* HTML, CSS, JavaScript

## ⚙️ How It Works

1. Captures live video from webcam
2. Detects face using computer vision
3. Predicts emotion using deep learning
4. Plays music based on detected emotion

## ▶️ Run Locally

```bash
git clone https://github.com/Kirankavala143/Emotion_Based_Music_Player.git
cd Emotion_Based_Music_Player
pip install -r requirements.txt
python app.py
```
